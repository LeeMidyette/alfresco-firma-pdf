Qué es

Alfresco firma pdf es un módulo para el sistema de gestión documental Alfresco que permite firmar electrónicamente documentos de tipo PDF mediante la herramienta @Firma.

Gracias la instalación de este módulo se puede usar Alfresco como un sistema de portafirmas, al permitir definir flujos que requieran la firma de documentos por los usuarios.


Descripción

Un portafirmas es una herramienta que permite definir procesos de firma de documentos entre usuarios. Los usuarios deben poder revisar los documentos del proceso y firmarlos si lo consideran oportuno, quedando el documento firmado disponible en la plataforma.

Alfresco dispone de las capacidades de gestión documental que permiten a una entidad que lo implante disponer de su documentación organizada y fácilmente accesible. Además facilita la creación de flujos entre usuarios sobre los documentos, pudiéndose convertir en un portafirmas con el módulo Alfresco firma pdf.

Este módulo añade a Alfresco:

  • Nueva acción disponible sobre los documentos: 'Firmar'.
  • Ventana de firma que utiliza @Firma como herramienta de firma.
  • Servicios de respositorio para generar nuevas versiones con el contenido firmado.
